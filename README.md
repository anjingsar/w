# xivwidget

A Discord profile widget application for FFXIV.
Displays various FFXIV stats on your Discord profile:

## Use
Authorize the bot here: https://discord.com/oauth2/authorize?client_id=1509844130082062396
Then run /widget setup in any channel. The bot will respond with further instructions to complete the setup.
